# Introdução

## Objetivo

Apresentar conceitos fundamentais de Aprendizado de Máquina, capacitando os estudantes a implementar algoritmos clássicos supervisionados e não-supervisionados, conduzir experimentos completos com dados reais ou simulados, avaliar modelos e interpretar resultados de forma crítica.

## Habilidades e competências

Ao final da disciplina, o estudante será capaz de:

- Implementar modelos supervisionados: k-NN, Árvores de Decisão, Random Forest, Gradient Boosted Trees, Regressão Linear, Perceptron e MLP.  
- Implementar métodos não-supervisionados: k-Means e clustering hierárquico.  
- Realizar análise exploratória e visualização de dados.  
- Avaliar modelos usando métricas de classificação e regressão.  
- Otimizar hiperparâmetros e interpretar resultados.  
- Usar ferramentas como Google Colab, GitHub, MLFlow e Docker para gerenciar experimentos.  

## Procedimentos didáticos

- Aulas expositivas dialogadas, debates e estudos de caso.  
- Dinâmicas práticas e apresentações de trabalhos.  
- Notebooks interativos via Google Colab e versionamento de código via GitHub.  

## Ementa

Introdução a Aprendizado de Máquina; análise exploratória de dados; métodos supervisionados (k-NN, Árvores, Random Forest, Gradient Boosted Trees, Regressão Linear, Perceptron, MLP); métodos não-supervisionados (k-Means e clustering hierárquico); avaliação de modelos; otimização de hiperparâmetros; prática de experimentos.
